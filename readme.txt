disaster engine version 5

c# raylib game engine by jazz "danger" mickle

setup:
for a good starting point, use the test content files from https://github.com/jmickle66666666/disaster_test
place the folder in a safe place, and modify disaster.cfg to point to it's location

for mac and linux, use dotnet core, go to the project directory (contains the src folder) and type `dotnet run`
